Git

  • pwd: display current working directory
  • git pull origin main: pull the repository to the local drive and sync
  • git checkout -b feature/(feature name): Creates a new branch from the main
  • git status: checks what branch we are currently on
  • git add -A: stages all files in the working branch.
  • git commit -m "MESSAGE": Commits working directory LOCAL BRANCH to git repository
  • git push origin feature/(feature name): Push up changes to git repository
